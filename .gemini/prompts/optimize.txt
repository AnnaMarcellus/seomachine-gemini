You are in OPTIMIZE mode. Perform comprehensive SEO audit and optimization.

## Task
Analyze the provided article and provide actionable recommendations to maximize search rankings while maintaining quality.

## Input
Article file path or content to analyze.

## Analysis Framework

### 1. Keyword Optimization Audit

#### Primary Keyword
- Identify target primary keyword
- Calculate density (target: 1-2%)
- Map all instances in:
  - H1 heading
  - First 100 words
  - H2 headings (2-3 should include it)
  - Body paragraphs (even distribution)
  - Conclusion
  - Meta elements

#### Semantic Keywords
- Identify LSI/semantic variations
- Check natural language usage
- Verify no keyword stuffing

#### Opportunities
- Headings where keywords could be added naturally
- Paragraphs that could benefit from keyword integration
- Alternative phrasing opportunities

### 2. Content Structure Audit

#### Heading Hierarchy
- Single H1 with primary keyword
- 4-7 H2 sections, logical progression
- Proper H3 nesting
- No skipped levels (H2→H4)

#### Content Organization
- Strong hook in introduction
- Each section delivers on promise
- Logical flow: Problem → Solution → Action
- Balanced section lengths
- Clear CTA in conclusion

#### Scannability
- Paragraph length: 2-4 sentences
- Lists for sequential items
- Bold/italics for emphasis
- White space between sections

### 3. Link Strategy Audit

#### Internal Links (Target: 3-5+)
- Count current internal links
- Evaluate relevance and context
- Check anchor text quality
- Identify missed opportunities

**Provide specific recommendations**:
- "In [Section], link to [Page] with anchor text '[suggested]'"

#### External Links (Target: 2-3+)
- Count authoritative external links
- Verify source credibility
- Check for broken links
- Identify claims needing citations

### 4. Technical SEO Audit

#### Meta Elements
- **Title**: 50-60 chars, includes primary keyword, compelling
- **Description**: 150-160 chars, clear value prop, CTA
- **URL Slug**: Concise, keyword-rich, lowercase with hyphens

Generate 3-5 improved alternatives for each.

#### Image Optimization
- Descriptive file names
- Alt text with keywords
- Strategic placement
- Recommend where images would help

#### Featured Snippet Opportunities
- Question-answer format
- List-based content
- Table structures
- Definition opportunities

### 5. Readability & UX

- Flesch Reading Ease score
- Grade level assessment
- Sentence length variety
- Transition words usage

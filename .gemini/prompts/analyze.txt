You are in ANALYZE mode. Analyze existing content for SEO health and performance.

## Task
Fetch and analyze content from a URL or file path to evaluate SEO performance, identify outdated information, and provide content health score.

## Input
URL or file path to analyze.

## Analysis Process

### 1. Content Fetch & Extraction
- Fetch content from URL or read from file
- Extract: title, meta description, headings, body content, links
- Note publish date (if available)

### 2. SEO Health Assessment

#### On-Page SEO (40 points)
- Title tag optimization (10 pts)
- Meta description (10 pts)
- Heading structure (10 pts)
- URL optimization (10 pts)

#### Content Quality (30 points)
- Word count vs. SERP benchmarks (10 pts)
- Readability scores (10 pts)
- Content freshness (10 pts)

#### Keyword Optimization (20 points)
- Primary keyword presence (10 pts)
- Keyword density (5 pts)
- Semantic keywords (5 pts)

#### Technical Elements (10 points)
- Internal links (5 pts)
- External links (5 pts)

### 3. Competitive Positioning
- Compare against current top 10 SERP results
- Identify content gaps
- Assess differentiation

### 4. Outdated Content Detection
- Check for outdated statistics
- Identify old examples/case studies
- Note deprecated tools/processes
- Flag changed best practices

### 5. Update Priority Assessment

**Score 0-100**:
- 90-100: Excellent, minor tweaks only
- 70-89: Good, some optimization needed
- 50-69: Needs work, significant updates required
- 0-49: Poor, major rewrite recommended

**Priority Levels**:
- **Critical**: High-traffic page with poor score
- **High**: Medium-traffic, moderate score
- **Medium**: Low-traffic but salvageable
- **Low**: Consider replacement rather than update

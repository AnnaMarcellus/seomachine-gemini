# SEO Guidelines

Search engine optimization requirements for all content.

## Keyword Strategy

### Primary Keyword
- **Density**: 1-2% of total word count
- **Placement**:
  - H1 heading (must)
  - First 100 words (must)
  - 2-3 H2 headings (recommended)
  - Conclusion (recommended)
  - Meta title (must)
  - Meta description (must)
  - URL slug (must)

### Secondary Keywords
- **Density**: 0.5% each
- **Placement**: Natural throughout content
- **Count**: 3-5 per article
- Include in at least one H2 or H3

### Semantic Keywords (LSI)
- Include naturally throughout
- Use variations and synonyms
- Don't force — should flow naturally

## Content Length

### Minimum Lengths by Content Type
- **Pillar page**: 3,000+ words
- **Standard blog post**: 2,000-3,000 words
- **Quick answer**: 1,000-1,500 words
- **Update/refresh**: Match or exceed previous length

### Quality Over Quantity
- Every word should add value
- Cut fluff and filler
- Depth > Length
- Match or exceed top 10 SERP average

## Heading Structure

### Hierarchy
```
H1: Only one per page, includes primary keyword
├── H2: Main sections (4-7 total)
│   ├── H3: Subsections
│   │   └── H4: Rarely needed
├── H2: Next main section
└── H2: Conclusion
```

### Best Practices
- Never skip levels (no H2 → H4)
- Use keywords naturally in headings
- Make headings descriptive
- One main idea per heading

## Link Strategy

### Internal Links
- **Target**: 3-5 per 1,000 words
- Link to relevant, high-value pages
- Use descriptive anchor text
- Spread throughout content
- Include in first and last sections

### External Links
- **Target**: 2-3 per 1,000 words
- Link to authoritative sources
- Reference statistics and data
- Open in new tab
- No-follow for untrusted sites

### Link Anchor Text
- Descriptive and natural
- Include keywords where appropriate
- Avoid "click here" or "read more"
- Vary anchor text for same URL

## Meta Elements

### Title Tag
- 50-60 characters
- Format: Primary Keyword | Brand
- Include primary keyword near start
- Compelling and click-worthy
- Unique for every page

### Meta Description
- 150-160 characters
- Include primary keyword
- Clear value proposition
- Call-to-action
- Unique for every page

### URL Structure
- Lowercase with hyphens
- Include primary keyword
- Remove stop words (a, an, the, etc.)
- Keep under 60 characters
- No special characters

## Featured Snippets

### Paragraph Snippets
- Answer question in 40-60 words
- Use "is," "are," "means" format
- Place after H2/H3 question

### List Snippets
- Use numbered or bulleted lists
- 5-9 items optimal
- Clear, parallel structure
- Start with action verbs

### Table Snippets
- Compare 3+ items
- 3-4 columns optimal
- Clear headers
- Consistent formatting

## Image Optimization

### File Names
- Descriptive, lowercase
- Include keywords
- Use hyphens (not underscores)
- Example: `website-speed-audit-results.png`

### Alt Text
- Describe the image
- Include keywords naturally
- 125 characters max
- Be specific

### Compression
- Use WebP when possible
- Compress PNGs/JPEGs
- Target <100KB per image
- Use lazy loading

## Technical SEO

### Page Speed
- Target <3 seconds load time
- Optimize images
- Minimize CSS/JS
- Use caching

### Mobile Optimization
- Responsive design
- Readable font sizes
- Touch-friendly buttons
- Fast mobile load time

### Schema Markup
- Article schema for blog posts
- FAQ schema for Q&A content
- HowTo schema for tutorials
- Review schema for comparisons

## Content Freshness

### Update Schedule
- **Pillar content**: Quarterly review
- **Trending topics**: Monthly review
- **Evergreen content**: Annual review
- **Statistics/data**: Update when new data available

### Refresh Signals
- Update publish date
- Add new section or examples
- Refresh statistics
- Improve formatting

## Tracking & Measurement

### Key Metrics
- Organic traffic
- Keyword rankings
- Click-through rate (CTR)
- Time on page
- Bounce rate
- Conversions

### Tools
- Google Search Console
- Google Analytics 4
- DataForSEO
- Ahrefs/SEMrush

## Common SEO Mistakes to Avoid

1. **Keyword stuffing** — Keep density natural
2. **Duplicate content** — Every page should be unique
3. **Missing meta descriptions** — Write for every page
4. **Broken links** — Check and fix regularly
5. **Slow page speed** — Optimize continuously
6. **Thin content** — Provide real value
7. **Ignoring search intent** — Match content to intent
8. **No internal linking** — Link strategically

# Style Guide

Grammar, formatting, and style standards for all content.

## Grammar & Punctuation

### Headings
- **H1**: Sentence case ("How to optimize your website speed")
- **H2**: Sentence case ("Why page speed matters")
- **H3**: Sentence case ("The impact on conversions")
- No periods at end of headings

### Lists
- Use parallel structure
- Start with action verbs for task lists
- End with periods for complete sentences
- No periods for fragments

### Numbers
- Spell out 1-9 ("five tips")
- Use numerals for 10+ ("12 strategies")
- Always use numerals for percentages, statistics, and measurements
- Use commas for thousands ("1,000 visitors")

### Dates & Times
- Month Day, Year ("January 15, 2024")
- 12-hour format with am/pm ("2:30 pm")
- Time zones when relevant

## Formatting

### Bold & Italics
- **Bold**: Key terms, important concepts, calls-to-action
- *Italics*: Emphasis, book/article titles, foreign words
- Avoid underlining (looks like links)

### Code & Technical Terms
- Inline code: `function_name()`
- Code blocks for multi-line examples
- Specify language for syntax highlighting

### Links
- Use descriptive anchor text (avoid "click here")
- External links open in new tab
- Internal links use relative paths

### Images
- Descriptive file names ("website-speed-audit-results.png")
- Alt text for accessibility
- Captions when helpful
- Compress before uploading

## Content Structure

### Introduction
- Hook in first 1-2 sentences
- State the problem or opportunity
- Promise what reader will learn
- 100-150 words

### Body Sections
- 300-500 words per H2
- 150-250 words per H3
- One main idea per section
- Transition between sections

### Conclusion
- Summarize key points
- Clear call-to-action
- 100-150 words

## Voice & Tone Consistency

### Do
- Write in second person ("you")
- Use active voice
- Be specific with data
- Show don't just tell

### Don't
- Use first person plural ("we think") unless necessary
- Write in passive voice
- Make vague claims
- Over-promote products

## SEO Formatting

### Meta Title
- 50-60 characters
- Include primary keyword near beginning
- Compelling and descriptive

### Meta Description
- 150-160 characters
- Include primary keyword
- Clear value proposition
- Call-to-action

### URL Slug
- Lowercase with hyphens
- Include primary keyword
- Remove stop words
- Keep under 60 characters

## Common Mistakes to Avoid

1. **Wall of text** — Break up long paragraphs
2. **Inconsistent formatting** — Follow these standards
3. **Missing transitions** — Guide readers between sections
4. **Weak conclusions** — End with clear next steps
5. **Over-optimization** — Keywords should feel natural

## Editing Checklist

Before publishing, verify:
- [ ] Headings follow sentence case
- [ ] Lists use parallel structure
- [ ] Numbers formatted correctly
- [ ] Links use descriptive text
- [ ] Images have alt text
- [ ] Meta elements optimized
- [ ] No spelling/grammar errors
- [ ] Consistent terminology
- [ ] Clear calls-to-action
- [ ] Proper formatting throughout
